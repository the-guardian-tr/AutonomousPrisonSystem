# Changelog

All notable changes to the **Autonomous Prison System** project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]
### Added
- **Robot Module**: ROS2 simulated patrol script (`robot/autonomous_patrol.py`).
- **DevOps**: Docker Compose orchestration and GitHub Actions CI workflow.

## [1.2.0] - 2025-12-14
### Added
- **Dashboard**: Professional Next.js UI with "Sci-Fi" cyberpunk theme.
- **AI Core**: Risk Engine implementation (`ai/risk_engine.py`) using Random Forest logic.
- **IoT**: Sensor node firmware simulation (`iot/sensor_node.cpp`).

### Changed
- Refactored `backend/main.go` to use `net/http` with structured API endpoints.
- Updated root README with professional branding, badges, and architecture diagram.
